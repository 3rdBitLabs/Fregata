# Code of Conduct

The Fregata community follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in Fregata's issues, discussions, or any other community space, you agree to abide by its terms.

If your concern is a security vulnerability rather than a conduct issue, see [SECURITY.md](SECURITY.md) instead.
